# OPC搭子 - 整体架构设计

> 产品名：OPC搭子
> 版本：v1.0 | 日期：2026-03-29

---

## 一、产品定位

**Slogan**：找到互补的人，少走弯路

**一句话**：OPC老板的互相帮助平台，帮助有技能的你找到互补的搭子。

---

## 二、产品结构

```
┌──────────────────────────────────────────────────────┐
│                    OPC搭子                            │
│                                                      │
│  ┌────────────┐  ┌────────────┐  ┌──────────────┐  │
│  │  微信H5    │  │  公众号    │  │   官网/PC    │  │
│  │  (核心产品) │  │  (内容+通知)│  │  (展示引流)  │  │
│  └────────────┘  └────────────┘  └──────────────┘  │
│                                                      │
└──────────────────────────────────────────────────────┘
```

### 三个入口的分工

| 入口 | 形态 | 作用 |
|------|------|------|
| **微信H5** | SPA应用，微信内打开 | 核心产品，搭子匹配 + 问答 + 社区 |
| **服务号** | 订阅号+服务号 | 内容发布（推文）+ 模板消息（合规提醒+通知） |
| **官网** | 展示页面 | SEO + 外部引流（知乎/小红书来的用户）|

### Nginx 配置

```nginx
# nginx.conf
server {
    listen 443 ssl http2;
    server_name opcdazi.com;

    ssl_certificate     /etc/nginx/certs/fullchain.pem;
    ssl_certificate_key /etc/nginx/certs/privkey.pem;

    # 前端静态文件
    location / {
        root /var/www/opcdazi/dist;
        try_files $uri $uri/ /index.html;
        expires 7d;
        add_header Cache-Control "public, immutable";
    }

    # 后端API
    location /api/ {
        proxy_pass http://127.0.0.1:8000;
        proxy_set_header Host $host;
        proxy_set_header X-Real-IP $remote_addr;
        proxy_set_header X-Forwarded-For $proxy_add_x_forwarded_for;
        proxy_read_timeout 60s;
    }

    # SSE实时通知
    location /sse/ {
        proxy_pass http://127.0.0.1:8000;
        proxy_set_header Host $host;
        proxy_buffering off;
        proxy_cache off;
        chunked_transfer_encoding on;
    }
}
```

## 九、docker-compose.yml

```yaml
version: '3.8'

services:
  nginx:
    image: nginx:alpine
    ports:
      - "80:80"
      - "443:443"
    volumes:
      - ./nginx/nginx.conf:/etc/nginx/nginx.conf
      - ./frontend/dist:/var/www/opcdazi/dist
      - ./certs:/etc/nginx/certs
    depends_on:
      - backend

  backend:
    build:
      context: ./backend
      dockerfile: Dockerfile
    environment:
      - DATABASE_URL=postgresql+asyncpg://opc:password@db:5432/opcdazi
      - REDIS_URL=redis://redis:6379/0
      - WECHAT_APP_ID=${WECHAT_APP_ID}
      - WECHAT_APP_SECRET=${WECHAT_APP_SECRET}
    depends_on:
      - db
      - redis
    restart: unless-stopped

  db:
    image: postgres:15-alpine
    environment:
      POSTGRES_DB: opcdazi
      POSTGRES_USER: opc
      POSTGRES_PASSWORD: ${DB_PASSWORD}

      - postgres_data:/var/lib/postgresql/data

      - "5432:5432"  # 本地调试用

    image: redis:7-alpine

    volumes:
      - redis_data:/data
    ports:
      - "6379:6379"  # 本地调试用
```
